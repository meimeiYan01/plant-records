# 多肉养殖记录应用

一个简单易用的多肉植物养殖记录网页应用，支持记录植物信息和养护过程。

## 功能特点

- ✅ 植物信息管理：添加、编辑、删除多肉植物
- ✅ 养护记录：记录浇水、施肥、换盆、修剪等养护活动
- ✅ 图片上传：支持为植物和养护记录上传图片
- ✅ 大图查看：点击图片可查看完整大图
- ✅ 响应式设计：适配电脑、平板和手机等多种设备
- ✅ 本地存储：数据保存在浏览器本地，无需网络

## 如何使用

### 在电脑上使用

1. 确保您的电脑已安装Python
2. 打开终端或命令提示符
3. 导航到应用所在目录
4. 运行以下命令启动本地服务器：
   ```
   python -m http.server 8000
   ```
5. 在浏览器中访问 `http://localhost:8000`

### 在手机上使用

1. 确保手机和电脑连接在同一个WiFi网络下
2. 查看您电脑的IP地址：
   - Windows：在命令提示符中运行 `ipconfig`，查找IPv4地址
   - Mac/Linux：在终端中运行 `ifconfig` 或 `ip addr`，查找本地IP地址
3. 在手机浏览器中输入 `http://电脑IP地址:8000`
   （例如：`http://192.168.1.100:8000`）

## 技术实现

- **前端框架**：纯HTML、CSS、JavaScript，无外部依赖
- **数据存储**：浏览器localStorage
- **响应式设计**：CSS Grid + Flexbox
- **图片处理**：File API + Base64编码
