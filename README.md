# 多肉养殖记录

🌵 简单好用的多肉植物养护记录工具，记录每一株多肉的成长故事 🌱

## 这是什么？

这是一个专门为多肉爱好者设计的网页应用，帮助您轻松记录和管理多肉植物的养护过程。无需安装，打开浏览器就能使用！

## 能做什么？

### 📋 管理您的多肉植物
- 添加新的多肉植物，记录名称、品种、入手日期和摆放位置
- 查看所有植物的基本信息
- 随时修改和更新植物信息
- 支持上传植物照片，用圆形头像展示

### 📝 记录养护过程
- 记录浇水、施肥、换盆、修剪等养护活动
- 为每条记录添加备注和照片
- 查看完整的养护历史
- 点击照片可查看大图

### 📱 随时随地使用
- 支持电脑、平板和手机访问
- 数据保存在您的浏览器中，无需网络
- 简洁美观的界面，使用流畅

## 如何使用？

### 方式1：直接访问（推荐）

如果您的朋友已经将应用部署到了网上，您只需要：
1. 打开浏览器
2. 在地址栏输入朋友分享的网址
3. 开始使用应用记录您的多肉植物

### 方式2：本地使用

如果您想在自己的电脑上使用：
1. 确保电脑已安装Python
2. 下载应用文件到电脑
3. 打开命令提示符（或终端）
4. 进入应用文件所在文件夹
5. 运行命令：`python -m http.server 8000`
6. 打开浏览器，访问 `http://localhost:8000`

## 功能说明

### 添加植物
1. 点击"添加多肉植物"按钮
2. 填写植物信息（名称、品种、入手日期、摆放位置）
3. （可选）上传植物照片
4. 点击"添加植物"完成

### 添加养护记录
1. 在植物卡片中点击"添加养护记录"按钮
2. 选择养护类型（浇水/施肥/换盆/修剪）
3. 填写日期和备注
4. （可选）上传养护照片
5. 点击"添加记录"完成

### 查看和修改植物信息
1. 在植物卡片中点击"编辑信息"按钮
2. 修改需要更新的信息
3. （可选）上传新照片替换旧照片
4. 点击"保存修改"完成


## 数据说明

- 所有数据保存在您的浏览器本地，只有您能看到
- 更换浏览器或清除浏览器数据会导致数据丢失
- 建议定期将重要数据备份到其他地方

## 常见问题

### 问：为什么不需要注册账号？
答：因为数据保存在您的浏览器中，所以不需要注册账号，更加隐私安全。

### 问：可以在多台设备上使用吗？
答：目前不支持多设备同步，数据只保存在当前使用的设备上。

### 问：照片会占用很多空间吗？
答：照片会压缩为Base64格式存储，一般情况下不会占用太多空间。

### 问：可以导出数据吗？
答：目前暂不支持数据导出功能，后续会考虑添加。
